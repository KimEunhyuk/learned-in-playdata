web API의 개요

참고

window 객체

JavaScript의 최상위 객체(모든 객체가 소속된 객체), 전역객체

BOM (browser object model)

브라우저 객체 모델, 브라우저 자체에 대한 몇 가지 조작이 가능, 브라저 별로 스펙이 다름

DOM (Document object model)

문서 객체 모델, 주로 html element의 조작을 통한 웹 페이지의 동적인 액션, 스타일링 등의 처리

JavaScript core(Object, Array, Function, ...)

JavaScript의 핵심 객체
